üöë Help! Common Issues

Problem: "Permission Denied" when pushing

Solution: You need to set up GitHub credentials

git config --global user.name "Your Name"

Problem: Site shows 404

Solution: Add an index.html file as your homepage 
cp ContinuumOS_Product_Showcase.html index.html
git add index.html
git commit -m "Add homepage"
git push
