2 . Quick‑Install Commands (Prototype Bundle)

Backend & Storage

# OCR & graph
brew install tesseract
wget https://neo4j.com/offline/community.zip
unzip community.zip

# Vector DB (Pinecone) – CLI
pip install pinecone-client

# Python API stack
python -m venv venv
source venv/bin/activate
pip install fastapi[all] langchain sentence-transformers openai uvicorn

Local Docker Compose (API + Ingestion)

# docker-compose.yml
version: "3.9"
services:
  api:
    build: ./api
    ports: ["8000:8000"]
    env_file: .env
    depends_on: [vector]
  vector:
    image: pinecone/server:latest
    ports: ["9000:9000"]

Front‑end

# React + WebRTC
npx create-next-app continuum-web
cd continuum-web
npm install simple-peer @unleash/proxy-client
npm run dev

Unity (VR)

# Unity Hub → install Unity 2022.3 LTS (URP template)
# In project:
# - Import Oculus Integration from Asset Store
# - Import Ready Player Me SDK (free)

Feature Flags (Unleash Cloud)

# Sign up at https://app.unleash-hosted.com
# Create Project: "Continuum"
# Add feature flag: feature.webrtc_chat (disabled by default)
# Copy proxy key into .env for React & Unity SDKs
